setCache

fun setCache(cache: Cache<T>, accessCount: Float): Unit (source)

set a Cache to this projector.

This must be invoked on the UI thread.

This will invoke onCacheUpdate to project the cache, and will increase accessCount of the Cache is increased immediately.

At some future time, the cache being updated, onCacheUpdate will be invoked again but it will not affect accessCount. accessCount increases only for the first time.
