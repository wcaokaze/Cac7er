getIfAlreadyLoaded

abstract fun getIfAlreadyLoaded(time: Long = System.currentTimeMillis(), accessCount: Float = 0.0f): T? (source)

returns the cached instance if it already exists on JVM memory, otherwise returns null.

time and accessCount only affect when the cached instance is returned.
