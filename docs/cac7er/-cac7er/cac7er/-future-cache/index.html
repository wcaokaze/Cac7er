FutureCache

interface FutureCache<out T> : Observable<T> (source)

Cache which denotes an empty file.

This can instantiate without any saving process, and can observe the file in which the content will be written at some future time.

fun fetchStatus(statusId: Long): FutureCache<Status> {
   launch(Dispatchers.IO) {
      // ...
      repository.save(statusId, status)
   }

   return repository.getFuture(statusId)
}
