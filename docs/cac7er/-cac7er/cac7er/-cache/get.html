get

abstract fun get(time: Long, accessCount: Float = 0.0f): T (source)

Parameters

time - the time when the Cache is accessed. Cac7er.gc makes this a criteria to consider how the Cache is important.

accessCount - the number of access. The type is Float for flexibility, infinite and negative values also work well if you can utilize it. But NaN is invalid (will be ignored).

Exceptions

ClassCastException - when the cached instance is not an instance of T. In the strict sense, ClassCastException is thrown by a cast operation complemented by the compiler.

Return
the cached instance
